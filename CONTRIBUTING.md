# 贡献指南

感谢你考虑为 requirement-clarifier 做出贡献。

## 许可证

本项目采用 **PolyForm Noncommercial License 1.0.0**，即非商用许可证。任何贡献（代码、文档、issue、PR 等）均视为接受本项目的许可证条款。这意味着：

- 你可以自由学习、修改、个人使用或内部使用。
- 你可以将修改后的版本用于非商业项目。
- **禁止将本项目或其衍生作品用于商业目的**（如作为付费服务、嵌入商业产品、用于商业工作流等）。

## 如何贡献

1. 先开一个 issue 描述你要解决的问题或要新增的能力。
2. 在 issue 中确认方案后再提交 PR。
3. 保持 SKILL.md 的风格：规则明确、避免过度工程、优先可执行性。

## 提交 PR 前请检查

- [ ] 没有新增不应提交的文件（如 `__pycache__/`）。
- [ ] 修改后的 SKILL.md 仍然符合现有文件结构。
- [ ] 脚本仍能正常运行（`python3 scripts/verify_evidence.py --help`）。
- [ ] 新增模板或清单文件被 SKILL.md 的“参考文件”部分引用。

## 不接受的贡献

- 违反项目许可证（商用）的修改。
- 与项目核心目标偏离的过度抽象或无关功能。
- 未经 issue 讨论的大型重构。
